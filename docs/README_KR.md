# 🎮 Mudspoon Tycoon (흙수저 타이쿤)

React + TypeScript + Vite로 제작된 웹 기반 인생 시뮬레이션 게임입니다.

## 📖 프로젝트 소개
플레이어는 캐릭터의 인생을 관리하며 매주 스케줄을 결정하고, 다양한 이벤트를 겪으며 성장해 나갑니다. 운과 선택이 중요한 역할을 하며, 최종적으로 어떤 인생을 살게 될지 결정하는 타이쿤 게임입니다.

---

## 🛠️ 기술 스택 (Tech Stack)
- **Frontend**: React (v19), TypeScript
- **Build Tool**: Vite (빠른 개발 환경)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Database**: Supabase (예정)
- **Deployment**: Vercel

---

## 📂 폴더 구조 안내
- `src/`: 소스 코드 (컴포넌트, 로직)
- `docs/`: 프로젝트 문서 (기획, 로그, 가이드)
  - `docs/00_Documentation_Guide.md`: 문서 작성 가이드
  - `docs/04_Dev_Log_and_Plan.md`: 개발 로그 및 일정 (Updated: Phase 2 Completed)

## 🚀 시작하기
1. 프로젝트 폴더 열기
2. 터미널에서 `npm install` (의존성 설치)
3. `npm run dev` (로컬 서버 시작)
