# 라이프 타이쿤 게임 메커니즘 (Life Tycoon Game Mechanics)

## 1. 행동 시스템 (Action System)
모든 행동은 **Power(기력)**를 소모합니다. 기본 비용(Base Cost)은 **10**입니다.
최종 비용은 **Stress(스트레스)**와 **Stamina(체력)**에 따라 동적으로 계산됩니다.

### 행동 효과 (Action Effects)
| 행동 (Action) | 비용 (Cost) | 스탯 변화 (Stat Changes) | 비고 (Notes) |
| :--- | :--- | :--- | :--- |
| **공부 (Study)** | 10+ | **지능(Int) +3**, **스트레스(Stress) +5** | 고위험 고수익. 시험 점수에 필수. |
| **운동 (Exercise)** | 10+ | **체력(Stm) +2**, **스트레스(Stress) +3** | 체력을 올려 행동 비용을 영구적으로 줄임. |
| **놀기 (Play)** | 10+ | **감각(Sen) +2**, **지능(Int) -1** | 감각을 올리지만 지능이 떨어짐. 스트레스 변화 없음. |
| **휴식 (Rest)** | 10+ | **스트레스(Stress) -10** | 스트레스를 해소하여 행동 비용 패널티를 제거. |

## 2. 동적 비용 공식 (Dynamic Cost Formula)
$$
\text{최종 비용} = 10 + (\text{스트레스 패널티}) - (\text{체력 보너스})
$$
*(최소 비용은 항상 1입니다)*

### 스트레스 패널티 (Debuff)
스트레스가 쌓일수록 행동 비용이 증가합니다.

| 스트레스 구간 | 패널티 | 상태 |
| :--- | :--- | :--- |
| **0 ~ 9** | 0 | 정상 (Normal) |
| **10 ~ 19** | +2 | 피로 (Tired) |
| **20 ~ 29** | +5 | 지침 (Exhausted) |
| **30 ~ 39** | **+10** | 번아웃 경고 (Burnout Warning) |
| **40 ~ 50** | **+20** | 건강 위험 (Health Collapse Risk) |

### 체력 보너스 (Buff)
체력이 높으면 더 적은 기력으로 행동할 수 있습니다.
- **공식**: `Math.floor(Stamina / 10)`
- 예시: 
  - 체력 20 -> 비용 -2 감소
  - 체력 50 -> 비용 -5 감소

---

## 3. 시험 점수 공식 (Exam Score Logic)
시험은 **5월, 7월, 10월, 12월** 시작 시(1주차) 치러집니다.
$$
\text{점수} = (\text{지능} \times 1.0) + (\text{이번 달 공부 횟수} \times 2.0) + (\text{운 } 0\sim10)
$$

## 4. 매력(Charm)과 친구 시스템 (Charm & Friends)
학생 시절의 감각(SEN) 스탯은 **'취업을 위한 대외활동/사회성'** 지표입니다.

### 감각(SEN)의 효과
1. **취업 조건**: **인턴(Intern, B급 기업)**에 지원하려면 **SEN 20 이상**이 필수입니다.
2. **친구 관계**: 친구 요청 거절 시 상대가 원한(Grudge)을 품으면 **SEN이 대폭 감소(-2~5)**합니다.

### 친구 시스템 (Friend System)
친구는 **등급(Rank)**이 나뉘며, 높은 등급일수록 강력한 효과를 가집니다.
모든 친구는 **"없는 것보다는 낫다"**는 철학을 가집니다. (D등급도 특정 상황에선 유용함)

#### 출현 확률 (Spawn Rates)
| 등급 (Rank) | 확률 (Rate) | 특징 |
| :--- | :--- | :--- |
| **S** | **1%** | **Game Changer**. 게임의 판도를 바꿈. |
| **A** | **4%** | **Specialist**. 특정 분야의 최강자. |
| **B** | **15%** | **Elite**. 확실한 이득 제공. |
| **C** | **30%** | **Normal**. 무난한 보너스. |
| **D** | **50%** | **Common**. 작지만 유용한 효과 or 등가교환. |

### 친구 시스템 (Friend System)
**등급(Rank) & 확률**: **S(1%) | A(4%) | B(15%) | C(30%) | D(50%)**
각 등급에는 4명의 친구 유형이 존재합니다. (총 20종)

#### 💖 여자친구 (Girlfriends) 메커니즘
- **1인 제한**: 여자친구는 동시에 1명만 사귈 수 있습니다.
- **High Risk, High Return**: 
  - **메리트**: 매주 **강력한 스트레스 해소** + **매력(Charm) +1** (연애 경험).
  - **디메리트**: **최대 기력(Max Power) 감소** (에너지 소모).
  - *결론: 행동 횟수는 줄지만, 스트레스 관리가 매우 쉬워져 "공부/알바"를 더 빡빡하게 돌릴 수 있음.*

#### 🌟 S등급: 미래 투자 (Future Investment)
S등급 친구들은 학생 시절에는 **아무런 이득이 없거나, 오히려 손해(비용/스트레스)**를 줍니다.
하지만 성인(사회인)이 되었을 때 **압도적인 보상**을 제공합니다.

## 5. 시간 시스템 (Time System)
- **시간 흐름 (Time Flow)**: **4초마다 1주(Week)**가 흐릅니다. (기본값)
- **시작 시점 (Start Date)**: **1년 1월 1주차**부터 시작합니다. (Year 1, Month 1)
- **일시정지 (Pause)**: 일시정지(`⏯️`) 상태에서는 시간이 흐르지 않으며, **활동(공부, 운동 등)도 수행할 수 없습니다.**

## 6. 이벤트 시스템 (Event System)
- **친구 요청 확률**: **매주 5%** 확률로 새로운 친구 요청이 들어옵니다.
  - 단, 친구가 0명인 경우 `1년 3월 2주차`에 **100% 확정** 이벤트가 발생합니다.

## 7. 노화 시스템 (Aging System)
매년 **1월 1일**, 캐릭터가 나이를 먹으며 신체 능력 저하로 체력이 감소합니다.

| 나이 구간 | 체력 감소량 (Stamina Decay) |
| :--- | :--- |
| **17 ~ 25세** | **-5** |
| **26 ~ 39세** | **-10** |
| **40세 이상** | **-20** |

> *나이가 들수록 건강 유지를 위해 꾸준한 운동이 필요합니다!*
