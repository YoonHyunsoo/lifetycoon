# 💰 수익화 모델 (Monetization Strategy) - 기획서

> **핵심 철학**: "광고는 방해가 아니라, 유저가 선택하는 **기회(Opportunity)**여야 한다."
>
> 게임의 몰입을 해치지 않으면서, 유저가 "아, 이건 광고 보고서라도 얻고 싶다!"라고 느끼게 만드는 **보상형 광고(Rewarded Ads)** 중심의 설계를 지향합니다.

---

## 1. 👼 부활 시스템 (The Second Chance)
**"죽음은 끝이 아니라, 천사의 투자를 받을 기회입니다."**

*   **배경**: 게임 오버(과로사, 파산) 시, 유저는 허탈감을 느낍니다. 이때 **단 한 번의 구제 기회**를 제공합니다.
*   **Trigger (발동 조건)**:
    *   **과로사 (Stress > 100)**: 병원에서 눈을 뜨며 의사가 "수술비가 필요한데..."라고 할 때.
    *   **파산 (Cash < 0)**: 사채업자가 찾아왔을 때, "수상한 투자자"가 등장.
*   **광고 보상**:
    *   생명력/스트레스 완전 회복 OR 빚 탕감 (0원으로 초기화).
    *   게임 계속 진행 가능 (이어하기).
*   **제약**: **인생(한 판) 당 1회 한정**. (긴장감 유지)

## 2. 📈 주식 시장 "내부자 정보" (Insider Info)
**"내년의 주가, 살짝 엿보시겠습니까?"**

*   **배경**: 주식 타이쿤의 핵심 재미는 '예측'입니다. 불확실성을 줄여주는 정보는 매우 강력한 유인책입니다.
*   **Trigger**: 매년 1월 `주식 거래소` 팝업 내부.
*   **UI**: "🕵️ 비밀 정보 입수" 버튼 (작게 배치).
*   **광고 보상**:
    *   **내년 급등/급락 종목 1개 힌트 제공**.
    *   예: "내년에는 [전자] 분야가 호황일 것입니다." (실제 확률 보정 적용)
*   **장점**: 강제성이 전혀 없으며, 전략적인 플레이를 원하는 유저가 스스로 선택함.

## 3. 🧧 "1+1" 더블 보상 (Double Trouble)
**"뜻밖의 횡재를 두 배로 불리세요!"**

*   **Trigger**:
    *   **복권 당첨 / 유산 상속 / 코인 대박** 등 **큰 금액(Large Cash)**을 획득했을 때.
    *   **연말 보너스** 수령 시.
*   **UI**: 획득 팝업 하단에 `[광고 보고 2배로 받기 x2]` 버튼 활성화.
*   **심리적 기제**: "이미 얻은 보상"을 잃기 싫어하는 손실 회피 심리 + 탐욕 자극.

## 4. ⚡ 긴급 스태미나/스트레스 케어 (Emergency Care)
**"이번 주를 헛되이 보낼 수 없다면?"**

*   **배경**: 중요한 시기(시험 직전, 승진 심사 전)에 스태미나가 부족하거나 스트레스가 꽉 찼을 때.
*   **Trigger**: 행동(공부/운동) 클릭 시, 자원 부족으로 실패했을 때 팝업.
*   **광고 보상**:
    *   **에너지 드링크**: 스태미나 +50 즉시 회복.
    *   **마사지 이용권**: 스트레스 -50 즉시 감소.
*   **제약**: 주(Month) 1회 제한. (게임 밸런스 붕괴 방지)

## 🛠️ 기술적 구현 가이드 (추후 적용)

### 1단계: AdMob / AdSense 연동 준비
*   웹 기반(React) 게임이므로 **Google AdSense (Web)** 또는 **AdMob (if App Wrap)** 사용.
*   현재는 웹 배포이므로 **Rewarded Interstitial (보상형 전면 광고)** 포맷이 적합.

### 2단계: 개발 로직 (Mockup)
```typescript
// 광고 시청 완료 콜백 예시
const handleWatchAd = (rewardType) => {
    showAd().then((success) => {
        if (success) {
            if (rewardType === 'REVIVE') revivePlayer();
            if (rewardType === 'DOUBLE') doubleIncome();
        }
    });
}
```

## 📝 요약
1.  **부활 (Revive)**: 가장 강력한 동기 부여. (불쾌감 최소화, 안도감 제공)
2.  **주식 힌트 (Stock Hint)**: 타이쿤 장르에 특화된 전략적 광고.
3.  **더블 보상 (Double)**: 긍정적인 상황에서의 기분 좋은 선택.

이 기획은 유저가 광고를 "귀찮은 것"이 아니라 **"게임 아이템"**처럼 느끼게 하는 데 초점을 맞추었습니다.
