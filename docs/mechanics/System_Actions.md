# ⚡ 행동 및 비용 (Actions & Costs)

## 1. 기본 규칙
*   모든 행동에는 **Power(기력)**가 듭니다.
*   **기본 비용**: 10 Power
*   하지만 내 상태(스트레스, 체력)에 따라 비용이 **할증**되거나 **할인**됩니다.

## 2. 최종 비용 공식
$$
\text{내야 할 기력} = 10 + (\text{스트레스 패널티}) - (\text{체력 보너스})
$$

### 😡 스트레스가 높으면? (비용 증가)
| 스트레스 | 추가 비용 |
| :--- | :--- |
| **0~9** | 없음 |
| **10~29** | +2 ~ +5 |
| **30~49** | +10 ~ +20 (효율 극악) |
| **50** | 💀 **과로사 (Game Over)** |

### 💪 체력이 좋으면? (비용 감소)
체력 10당 비용이 1씩 줄어듭니다.
*   체력 30이면? -> 비용 -3 할인.
*   체력 90이면? -> 비용 -9 할인 (공짜에 가까움!)
