# 👪 부모님 퀘스트 (Parent Quests)

## 1. 개요
부모님은 플레이어의 인생에 끊임없이 개입하며 미션(Quest)을 줍니다.
귀찮을 수도 있지만, 성공하면 집을 사주실 수도 있습니다.

## 2. 발생 주기
*   기본적으로 **2~5년**마다 랜덤하게 발생합니다.
*   **운(Luck)**이 좋으면 부모님이 더 자주 찾아와서 용돈을 주십니다.

## 3. 주요 퀘스트 목록
| 시기 | 퀘스트명 | 조건 | 보상 (성공 시) |
| :--- | :--- | :--- | :--- |
| **학창시절** | **성적 우수** | 시험 성적 상위 10% | 고액 과외(지능↑) / 최신 폰 / 용돈 |
| **취준생** | **취업 성공** | 연내 취업 성공 | 정장 선물(매력↑) / 보약(체력↑) / 용돈 |
| **사회인** | **결혼/출산** | 연내 결혼 또는 손주 보기 | **주택 자금 지원 (1억~5억)** 🏠 |
| **전구간** | **건강 관리** | 1년간 병원 안 가기 | 특제 보약 (체력 영구 증가) |

## 4. 효도의 중요성
부모님 퀘스트를 무시하면 불효자(Reputation 하락)가 될 수 있습니다.
가능하면 효도하세요. 자산 증식의 지름길입니다.
