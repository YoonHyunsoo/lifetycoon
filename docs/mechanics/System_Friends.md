# 🤝 친구 시스템 (Friend System)

## 1. 친구의 중요성
인생은 혼자 사는 게 아닙니다. 친구는 각종 버프(Buff)와 도움을 줍니다.

## 2. 만남과 이별
*   **만남**: 매주 **5%** 확률로 새로운 친구가 접근합니다.
*   **슬롯**: 최대 2명까지만 사귈 수 있습니다. (3명째부터는 교체해야 함)
*   **거절 페널티**: 친구 요청을 거절하면 마음의 상처를 입어 **매력(Charm)**이 깎입니다.

## 3. 친구 등급 (Tier)
| 등급 | 등장 확률 | 특징 | 예시 |
| :--- | :--- | :--- | :--- |
| **S** | **1%** | 인생 역전 | **재벌 3세**: 유지비가 들지만 나중에 사업 자금을 대줌. |
| **A** | **4%** | 전문가 | **전교회장**: 모든 스탯 상승 효율 증가. |
| **B** | **15%** | 엘리트 | **금수저**: 매달 용돈을 줌. |
| **C** | **30%** | 일반인 | **인싸**: 매력을 올려줌. |
| **D** | **50%** | 소시민 | **베짱이**: 휴식 효율을 높여줌. |

## 4. 여자친구 (Girlfriends)
*   **효과**: 매주 스트레스를 엄청나게 줄여줍니다.
*   **비용**: 데이트하느라 기운이 빠져서 **최대 체력(Max Power)**이 줄어듭니다.
*   **전략**: "짧고 굵게" 공부하고 싶을 때 유용합니다.
