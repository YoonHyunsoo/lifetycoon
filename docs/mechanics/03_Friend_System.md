# 🤝 친구 및 인간관계 (Friend System)

## 1. 개요
인생에서 '인맥'은 중요한 자산입니다. 친구는 랜덤하게 찾아오며, 수락하거나 거절할 수 있습니다.

## 2. 친구 사귀기
- **발생 확률**: 매주 **5%** (랜덤).
- **확정 이벤트**: 친구가 0명일 경우, **1년 3월 2주차**에 100% 확률로 이벤트 발생.
- **조건**: **인싸(Influencer)** 유형 등 일부 친구는 일정량의 **매력(Charm)**을 요구할 수 있습니다.

## 3. 리스크 & 리턴
친구 요청을 거절하면 **상처(Grudge)**를 받아 내 **매력(Charm)**이 대폭 깎일 수 있습니다.
- 거절 시 패널티: **매력 -2 ~ -5**

---

## 4. 친구 등급표 (Tier List)
친구는 5등급(S~D)으로 나뉩니다.

| 등급 | 확률 | 설명 |
| :--- | :--- | :--- |
| **S** | **1%** | 인생 역전급 (재벌 3세, 정계 자녀 등). 유지비가 많이 들지만 보상이 확실함. |
| **A** | **4%** | 각 분야 전문가 (창업가, 전교회장). |
| **B** | **15%** | 엘리트 친구들 (금수저, 모범생). |
| **C** | **30%** | 일반적인 친구들 (겜돌이, 인싸). |
| **D** | **50%** | 소소한 친구들 (매점 친구 등). |

## 5. 여자친구 (Girlfriends)
- **제약**: 동시 1명만 교제 가능.
- **특징**: 매주 **스트레스를 대폭 줄여주지만**, 데이트 비용으로 **최대 기력(Max Power)**을 영구적으로(사귀는 동안) 깎아먹습니다.
  - *전략: 기력 총량은 줄지만, 휴식 횟수를 줄일 수 있어 효율적일 수 있습니다.*
