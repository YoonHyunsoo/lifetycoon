# 📝 시험 로직 (Exam System)

## 1. 시험 일정
학창 시절(17~19세) 동안 매년 **4회** 치러집니다.
*   **5월**: 중간고사
*   **7월**: 기말고사
*   **10월**: 중간고사
*   **12월**: 기말고사

## 2. 점수 산출 공식
시험 점수는 '평소 실력(지능)'과 '벼락치기(노력)', '운'의 결합입니다.

$$
\text{최종 점수} = (\text{지능} \times 1.0) + (\text{이번 달 공부 횟수} \times 2.0) + (\text{운 } 0\sim10)
$$

*   **지능(Int)**: 베이스 점수. 평소에 책을 많이 읽어야 합니다.
*   **공부 횟수**: 시험이 있는 달(Month)에 수행한 '공부하기' 액션 횟수입니다. 벼락치기 효과가 매우 큽니다.

## 3. 등급 및 보상
| 백분위 | 등급 | 보상 |
| :--- | :--- | :--- |
| **상위 5%** | **1등급 (전교권)** | 장학금 + **학자금 대출 면제권** (나중에 대학 공짜) |
| **상위 10%** | **2등급 (우수)** | 용돈 보너스 |
| **하위 50%** | **5등급 이하** | 부모님의 잔소리 (스트레스 증가) |
