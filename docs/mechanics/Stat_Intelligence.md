# 🧠 지능 (Intelligence)

## 1. 정의
학업 성취도, 논리적 사고, 정보 분석력을 나타내는 수치입니다.

## 2. 역할
*   **시험 성적**: 중간/기말고사 점수 산출의 기본 베이스(Base)가 됩니다.
*   **주식 정보**: 지능이 높으면 주식 시장에서 '작전주'나 '리스크'를 파악할 확률이 높아집니다.
*   **업무 성과**: 직장인(사무직)의 경우 업무 성공률에 기여합니다.

## 3. 획득 및 소모
*   **[증가] 공부하기**: +3 (기본값)
*   **[감소] 놀기**: -1 (과도한 유흥 시)
