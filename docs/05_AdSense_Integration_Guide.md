# 구글 애드센스(AdSense) 연동 가이드

현재 게임에는 **모의 광고 시스템(Mock Ad System)**이 구현되어 있습니다. 실제 수익 창출을 위해 구글 애드센스를 연동하려면 다음 절차를 진행해야 합니다.

## 1단계: 배포 (Deployment)
애드센스 승인을 받으려면 **실제로 접속 가능한 웹사이트**가 있어야 합니다.
1. **GitHub Desktop**에서 변경사항을 Commit & Push 합니다.
2. **Vercel**에서 배포가 완료되었는지 확인합니다.
3. 배포된 URL (예: `life-tycoon.vercel.app`)로 접속하여 게임이 정상 작동하는지 확인합니다.

## 2단계: 도메인 구매 (권장 사항)
구글 애드센스는 `vercel.app` 같은 하위 도메인보다 `lifetycoon.com` 같은 **개인 도메인**에서 승인될 확률이 훨씬 높습니다.
- 가비아, 고대디 등에서 도메인을 구매하여 Vercel에 연결하는 것이 좋습니다.
- *필수는 아니지만, 승인 거절 확률을 줄이기 위해 권장합니다.*

## 3단계: 애드센스 신청 (사용자 진행)
1. [구글 애드센스](https://adsense.google.com/)에 접속하여 가입합니다.
2. **사이트 추가**를 클릭하고 배포한 URL을 입력합니다.
3. **애드센스 코드(Script Snippet)**가 제공됩니다.
   - **중요:** 이 코드를 복사하여 저에게 알려주세요. (혹은 `index.html`의 `<head>` 태그 안에 직접 넣어야 합니다.)

## 4단계: 기술적 연동 (개발자 진행)
사용자분이 제공해주신 Publisher ID(`pub-9382505899208273`)로 연동 설정을 완료했습니다.
1. ✅ **스크립트 삽입**: `index.html`에 애드센스 연동 스크립트 추가 완료.
2. ✅ **Ads.txt 추가**: `public/ads.txt` 파일 생성 및 설정 완료.
3. ✅ **리워드 광고 구현**: `AdSenseBanner` 컴포넌트에 해당 ID가 적용되었습니다.
   - *참고: 웹용 애드센스 게임 광고(H5 Games Ads)는 별도의 승인 절차가 필요할 수 있습니다.*

## 5단계: 검토 및 승인
- 모든 설정이 완료되면 구글의 검토를 기다립니다 (통상 2주 소요).
- 승인이 완료되면 게임 내에서 실제 광고가 송출되고 수익이 발생합니다.
